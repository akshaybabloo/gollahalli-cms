# Gollahalli Website

## 1 Django Template Tags

These are the tags that were used in this project:

1. To get name
  ```python
  {{ content.get_name }}
  ```
2. To get Twitter
  ```python
  {{ content.get_twitter }}
  ```
3. To get GitHub
  ```python
  {{ content.get_github }}
  ```
4. To get Linkedin
  ```python
  {{ content.get_linkedin }}
  ```
